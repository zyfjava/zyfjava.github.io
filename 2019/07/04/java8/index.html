发表于 2019-07-04 | 分类于 java

Java 8 (又称为 jdk 1.8) 是 Java 语言开发的一个主要版本。 Oracle 公司于2014 年 3 月 18 日发布 Java 8 ，它支持函数式编程，新的 JavaScript 引擎，新的日期 API，新的Stream API 等。Java8 新增了非常多的特性，我主要进行记录一下几个:Lambda 表达式、方法引用 、默认方法。
