初探Activiti 6.0

发表于 2019-06-27 | 分类于 java、activiti


activiti介绍 Activiti是由Alfresco软件在2010年5月17日发布的业务流程管理(BPM)框架,它是覆盖了业务流程管理，工作流，服务协作等领域的一个开源，灵活的，易扩展的可执行流程语言框架。本次我使用Activiti6.0来做一个例子。

一、简介


我的第一个Activiti小例子是一个最常见的公司内部的电话费报销申请流程：需要报销的员工申请报销后由员工所在部门的部门经理进行审批，通过后再交由管理本部门的公司副总进行审批，通过后最后交由董事长进行审批，这其中除了普通员工可以取消订单申请结束流程之外，其他的领导在驳回请求后都会返回到电话费申请任务，但是由于本例子未涉及到驳回在申请的业务流程因此暂时只是理论阶段。（这个流程很简单，但是需要注意的是员工提交申请后的第一级审批是部门经理，但是但是第二级的公司副总是同时管理了多个部门并且一个公司含有多个副总）。本例子使用了springboot2.0+Activiti6.0+mysql+thymeleaf,持久层选用了mybatis的项目整体结构见下图：
